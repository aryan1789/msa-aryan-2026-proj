# /specs — Planning, Design & AI-Assisted Development

This folder is required evidence for the MSA 2026 Phase 2 assessment. It documents how
this project was planned, the design decisions behind it, and how AI was used during
development.

## Contents

| File | Purpose |
|------|---------|
| `00-project-plan.md` | The overall project plan — concept, tech stack, features, weekly breakdown |
| `01-design-decisions.md` | A log of key decisions and *why* each was made (feeds the video's "design decisions" section) |
| `02-ai-usage-log.md` | A running record of how AI was used, including prompts and how outputs were critically evaluated |

## How this folder is maintained

These are **living documents**, updated throughout development — not written all at once at
the end. New design decisions and AI interactions are appended as they happen, so this folder
is an honest record of the actual development journey.
